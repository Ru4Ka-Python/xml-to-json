Описание

Данный проект представляет собой Python-скрипт, предназначенный для преобразования файлов из формата XML в формат JSON.

Установка
Для корректной работы требуется установленный Python версии 3.11 или выше.
После установки Python необходимо запустить BAT-файл, который автоматически установит все библиотеки и зависимости, необходимые для функционирования скрипта.

Использование
1. Выберите XML-файл для конвертации.
2. Выберите режим работы:
   - Полный (стандартный) — конвертирует исходный XML-файл в один JSON-файл.
   - Разделённый — разбивает содержимое XML-файла на отдельные страницы и сохраняет их в виде нескольких JSON-файлов. Данный режим рекомендуется для обработки крупных файлов и использования в задачах обучения ИИ или ботов.
3. Укажите директорию для сохранения результирующих файлов.
